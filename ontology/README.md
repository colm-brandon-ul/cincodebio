## Instructions for managing Ontology IRI

The latest (stable version) ontology is stored in the main branch for the cinco de bio repository which equates to the **ontologyIRI** whose value is the url below (with main substituted for <branch_name>). However, along with storing the stable version in the main branch, a new branch should be created for each new ontology version, which <branch_name> is substituted for the following the **format** - ***ontology-vX.X.X***, subtituting X's for some incremental semantic version values, the url of which equated to **owl:versionIRI** in the ontology.

"https://raw.githubusercontent.com/colm-brandon-ul/cincodebio/`<branch_name>`/ontology/cincodebio.owl"
